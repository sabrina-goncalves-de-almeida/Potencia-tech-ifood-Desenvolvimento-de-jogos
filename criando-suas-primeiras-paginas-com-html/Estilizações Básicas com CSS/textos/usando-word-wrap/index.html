Propriedade word-wrap

Define, caso uma palavra seja grande demais para caber em uma linha, se ela será quebrada em qualquer caractere para que uma parte dela fique na linha e o restante na próxima, ou se não, toda palavra deve ficar na mesma linha e ultrapassar os limites que definimos para o elemento.

Exemplos:

word-wrap: normal;
