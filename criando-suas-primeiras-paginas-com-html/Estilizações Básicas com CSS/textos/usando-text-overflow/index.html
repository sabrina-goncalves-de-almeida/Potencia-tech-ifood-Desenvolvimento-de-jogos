Propriedade text-overflow

Define como um texto será exibido caso exista mais texto no elemento que não está aparecendo

Exemplos:

text-overflow: clip;

Chega de saudade A realidade é que sem ela não há paz

text-overflow: ellipsis;

Chega de saudade A realidade é que sem ela não há paz

text-overflow: string;

Chega de saudade A realidade é que sem ela não há paz
